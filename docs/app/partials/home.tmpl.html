What is AngularJS Material?

AngularJS Material is both a UI Component framework and a reference implementation of Google's Material Design Specification. This project provides a set of reusable, well-tested, and accessible UI components based on Material Design.

What about Angular?

AngularJS Material recently released Version 1 which we consider to be stable and ready for production use. Developers should note that AngularJS Material works only with AngularJS 1.x.

  • Current AngularJS Material development efforts are focused on bug fixes and minor improvements.
  • AngularJS Material development is also in-progress at the angular/material2 GitHub repository.

Please refer to our changelog for up-to-date listings of all v1.x improvements and breaking changes.
