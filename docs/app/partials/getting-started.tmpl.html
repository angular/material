Include Optional Dependencies

Angular Material integrates with some additional libraries which you may optionally include:

  • ngMessages - Provides a consistent mechanism for displaying form errors and other messages.
  • ngSanitize - The ngSanitize module provides functionality to sanitize HTML content in Material components.
  • ngRoute - Provides a clean routing system for your application.

Unsupported Integrations

Angular Material v1.0 has a few known integration issues with the following libraries:

  • ngTouchs - This module interferes with ngMaterial's click, tap, and swipe support for touch-enabled devices.
  • Ionic - Open-source SDK for developing hybrid mobile apps with Web technologies has touch support that interferes with ngMaterial's mobile gesture features.
