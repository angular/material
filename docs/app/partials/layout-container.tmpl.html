Overview

Angular Material incorporates a responsive CSS layout based on flexbox, rather than the traditional CSS floats. With flexbox now available in a majority of browsers, flexbox is clearly the best bet.

The layout system is based upon element attributes rather than CSS classes. Attributes provide an easy way to set a value eg `layout="row"`, and additionally helps us separate concerns: attributes define layout and classes define styling.

Layout Attribute

Use the layout attribute on an element to align its children horizontally in a row (layout="row"), or vertically in a column (layout="column").

Responsive Layout

Defining different layouts on different screen sizes as simple as adding a new attribute. The layout attribute defines the default layout. Use a more specific attribute to define the layout on specific devices:

  • layout sets the default layout on all devices.
  • layout-sm sets the layout on devices <=600px wide (phones).
  • layout-gt-sm sets the layout on devices >600px wide (bigger than phones).
  • layout-md sets the layout on devices >600px and <=960px wide (tablets in portrait).
  • layout-gt-md sets the layout on devices >960px wide (bigger than tablets in portrait).
  • layout-lg sets the layout on devices >960 and <=1200px wide (tablets in landscape).
  • layout-gt-lg sets the layout on devices >1200px wide (computers and large screens).
