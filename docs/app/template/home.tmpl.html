Material Design is a specification for a unified system of visual, motion, and interaction design that adapts across different devices.

The material design project for Angular is a complementary effort to the Polymer project's paper elements collection. Our goal is to provide a set of AngularJS-native UI elements that implement the material design system. This demo is a work in progress.
